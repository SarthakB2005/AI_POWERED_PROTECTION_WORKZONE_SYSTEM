# 💻 Software & Tools Used

This document lists all the software, tools, and libraries used to build this project.

---

## 🧠 Programming & Development

### 1. Python
- Used for AI model and PPE detection
- Version: 3.x
- Link : https://www.python.org/

---

### 2. STM32CubeIDE
- Used to program STM32F103 microcontroller
- Used for embedded C development
- Link : https://www.st.com/content/st_com/en/stm32cubeide.html

---

## 🤖 AI & Computer Vision

### 3. ROBOFLOW
- Open source platfrom which allows to use already made ML model
- Easily download model offline and run it offline
- Link : https://roboflow.com/

---

### 4. STMCubeProgrammer
- Used to run the STM code written
- Upload code to STM32 for final working
- Link : https://www.st.com/en/development-tools/stm32cubeprog.html
